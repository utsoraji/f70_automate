4  使用上の制限・注意

4.1  制限事項

●  アクセス可能な「WAVE  LOGGER  X」は、オートメーションから起動したオブジェクトのみです。起動

済みの「WAVE LOGGER X」に対してアクセスすることはできません。

●  サンプリング数を指定して収集した場合、トリガ発生、プリトリガ状態からトリガ待ちへの遷移、計測

終了などの通知を受け取ることはできません。

●  エラー発生の通知を受け取ることはできません。状態を定期的に監視してください。

●  収集条件の設定（変更）はできません。保存された設定ファイルを読み出すことは可能です。複

数の設定を切り換える場合は、設定ファイルを事前に「WAVE  LOGGER  X」で作成し、読み出し

てください。

●  CAN  および  CANFD  ユニット設定で、CSV  保存した設定ファイルは読み出せません。NR-C512

CAN  データ収集ユニット、または  NR-CF512  CANFD  データ収集ユニット  を使用する場合は、

WAVE  LOGGER  X でファイルから設定データ保存を用いて保存した設定ファイル（拡張子 ycf、

xcf）を読み出してください。

●  設定ファイルの  Excel 転送設定は OFF にしてください。Excel へのデータ書き込みはオートメーシ

ョンでデータを取得し、Excel へ書き込んでください。

●  計測データ読み出しで取得できる値は、スケーリング／スムージング／物理量変換前の値です。  ま

た、演算チャンネルのデータは取得することはできません。

●  収集開始日時、トリガ発生日時、データ先頭日時を取得することはできません。

●  モニタ機能、設定調整機能は使用できません。

●  オートメーション経由で NR-PM10 と NR-PM01 の設定は出来ません。

●  WAVE LOGGER X に接続されているカメラの画像を取得することは出来ません。

4.2  収集条件

●

リピート収集は、「シングル収集を繰り返す」ように制御してください。特に、一度の収集が短時間で

終了する場合は、何度目に収集されたデータなのか確認ができません。

●  連続収集時は、「保存が終わったファイルを開いてデータを取得する」ように制御してください。直接

データを取得した場合、保存周期ごとにデータが更新されるため、切り換わり部のデータを取得でき

ないことがあります。

4.3  オートメーション例外

●  異常終了の場合、戻り値でエラーコードを返す関数もありますが、内容によってはオートメーション例

外をスローします。

●  オートメーション例外がスローされるケースの例を、以下に示します。

●  閉じられているドキュメントにアクセスしようとした場合

●  引数の型が異なる場合（VARIANT 型の場合）

5  起動と初期化

◼  サーバオブジェクト名は以下を使用します。

WAVE LOGGER X （NR-X100 シリーズ）：「WaveLoggerX.Application」

WAVE LOGGER X 体験版（NR-X100 シリーズ）：「WaveLoggerXDemo.Application」

◼

「WAVE LOGGER PRO」、「WAVE LOGGER」のサーバオブジェクト名は使用できません。

◼  オートメーションで起動した「WAVE  LOGGER  X」に対してのみ、操作できます。起動済みの「WAVE

LOGGER X」に対して操作することはできません。

◼  起動時の対応接続機器  ID  は、環境設定に関係なく「未設定」の状態になっています。NR-X100（NR-

いない場合、収集保存されたファイルを開いて確認することはできますが、収集することはできません。

9  サンプルプログラムについて

サンプルプログラムは、C:\program  files\KEYENCE\WAVE  LOGGER  X\samples\Automation.zip

ファイルに入っています。zip ファイルはローカルの作業ディレクトリに展開してください。zip ファイルを展開すると、

VBA  用のサンプルプログラムは\VBA\sample*  フォルダ、C#用のサンプルプログラムは  \Csharp\sample*

フォルダ、Python 用のサンプルプログラムは\Python\sample*フォルダに入っています。

サンプルプログラムを利用するときは、実運用前に使用用途に沿った動作になっているかを確認の上、使用してくだ

さい。

■  サンプル 1

最も基本的な操作例です。

「ゼロ点調整」～「収集開始 / 停止」をおこないます。

■  サンプル 2

収集データを取得し、平均を演算する操作例です。

収集データを配列の中に取り込み、演算をおこなって、C#では CSV ファイルに、Python ではコンソール(標

準出力)に、VBA では Excel ワークシートに書き込みます。
